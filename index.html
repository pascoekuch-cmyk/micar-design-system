03 / Components

Components

50+组件覆盖导航、数据录入、数据展示、反馈四大类。按钮圆角6px，弹窗420/600px，输入框高度36px。

Button 按钮

Base 6px圆角，Pill 1000px圆角。高度 36px / 28px / 24px 三档。支持 Primary、Secondary、Ghost、Danger、Success、Warning 六种风格。

Button Sizes 按钮尺寸

三档高度：Default 36px、Small 28px、Mini 24px。内边距 12px(水平) / 4px(垂直)。

Pill Button 药片按钮

圆角 1000px，完全药片形。适用于标签、筛选器等场景。

Functional Buttons 功能按钮

Success #00B227、Danger #FF4D4D、Warning #FF6A00，每色10级色阶。

Input 输入框

高度 36px，圆角 6px。支持 Form、Table、验证码、数字输入四种变体。Focus 态使用品牌色边框 + 光晕。

07 / Guidelines

Do's and Don'ts

使用小米设计系统时的关键准则。

Do

使用 MiSans 作为所有文本的基础字体
使用 Design Token 管理颜色、间距、圆角
基础按钮圆角保持 6px，药片按钮使用 1000px
填充交互状态使用 #191C24 的透明度变化（5%/12%/16%）
功能色使用对应的10级色阶
使用 fontSystem/{size} {weight} 的Token命名规范
阴影使用 S1/S2/S3 三级体系

Don't

不要使用纯黑色 #000000 作为阴影或文本色
不要跳过 Design Token 直接使用硬编码颜色值
不要混用 MiSans 和 MiType（MiType 仅用于数字）
不要忽略填充状态的透明度梯度
不要使用过大或过小的圆角
不要忽略车载大屏的字号适配（24px–88px）
